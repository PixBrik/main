# Meshy-6 Library Generation Batch — Animals, Cars & Hero Gifts

The categories no free CC0 source can fill at PixBrik's quality bar. Run
each prompt in **Library Studio → GENERATE MASTER** (opened from the
backoffice: Model Library → OPEN SECURE LIBRARY STUDIO). Every master =
two paid stages (preview + refine, ≈15–20 credits). The full list below is
~35 masters ≈ **550–700 credits** (account held ~1,110 at last check) — run
in priority order and stop whenever the balance says so.

**Prompt recipe that works for brick conversion:** subject + pose +
`studio product photography, centered, neutral gray background, even
lighting, full body, no base, no text`. Avoid: brands, logos, real car
model names (protected trade dress), busy bases/dioramas, extreme thin
protrusions (whiskers, thin antennas — they vanish below one brick).

After each generation: inspect the raw 3D, check the brick proposal reads
clearly at BALANCED, then publish with the category noted below. If a
result disappoints, regenerate with a pose tweak before spending on the
next subject.
